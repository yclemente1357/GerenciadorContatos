📌 Gerenciador de Contatos

Este projeto foi desenvolvido com o objetivo de praticar e consolidar conceitos fundamentais de JavaScript, com foco em manipulação do DOM, lógica de programação e operações com arrays.

A interface (HTML e CSS) foi utilizada apenas como suporte visual, enquanto toda a lógica da aplicação foi implementada em JavaScript.

🚀 Funcionalidades

O sistema permite:

➕ Cadastrar novos contatos
📋 Listar contatos dinamicamente na tela
✏️ Editar informações de um contato
❌ Excluir contatos da lista
🔍 Filtrar contatos em tempo real pelo nome
🧠 Conceitos aplicados

Durante o desenvolvimento, foram utilizados conceitos importantes como:

Manipulação do DOM (getElementById, innerHTML)
Eventos (onclick, addEventListener)
Estruturas condicionais (if, operadores lógicos)
Arrays e métodos:
push
splice
filter
forEach
Template strings (` `)
Controle de estado (lista de contatos e filtro)
⚙️ Como funciona

Ao preencher os campos e cadastrar um contato:

Os dados são armazenados em um array
A lista é renderizada dinamicamente na tela
Cada item possui ações de edição e exclusão
O campo de busca permite filtrar os contatos em tempo real, sem alterar a lista original
🎯 Objetivo do projeto

O principal objetivo deste projeto foi ganhar domínio prático em JavaScript, simulando funcionalidades comuns em aplicações reais, como um CRUD simples e filtragem dinâmica de dados.
